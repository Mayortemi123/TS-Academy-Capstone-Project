# TS-Academy-Capstone-Project
Capstone project notebooks and datasets from Temi Mayowa .A
Customer Churn Prediction Project
📌 Overview

This project focuses on predicting customer churn for a bank using machine learning techniques. The goal is to identify customers at risk of leaving and provide insights to support customer retention strategies.


📊 Dataset
Source: (Kaggle / provided dataset)
Records: ~10,000 customers
Features: 14 variables including:
Age
Balance
Geography
Tenure
Number of products
Active membership status




🎯 Problem Statement

Customer churn leads to significant revenue loss for businesses. This project aims to build a predictive model that can identify customers likely to churn, enabling proactive retention efforts.


⚙️ Data Preprocessing
Handled missing values
Encoded categorical variables (Label Encoding / One-Hot Encoding)
Feature scaling was applied where necessary
Addressed class imbalance (~80% non-churn vs 20% churn)


🤖 Models Used
Logistic Regression
Decision Tree
Random Forest


🔍 Key Insights
Customers with low activity levels are more likely to churn
Higher account balance correlates with churn risk
Customers with fewer products show higher churn probability


💡 Business Impact
Enables banks to identify high-risk customers early
Supports targeted retention strategies (offers, engagement)
Reduces revenue loss from customer attrition
